# Medical Insurance Cost Prediction
A machine learning web application that predicts medical insurance costs based on user details such as age, BMI, smoking status, number of children, gender, and region. The project uses a Linear Regression model trained on a medical insurance dataset and is deployed using Streamlit.



## Features
* Predict insurance charges instantly
* Interactive Streamlit web application
* Data preprocessing and feature encoding
* Exploratory Data Analysis (EDA)
* Linear Regression model implementation
* Real-time prediction system


## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit
* Joblib



## Dataset
The project uses the Medical Insurance Dataset from Kaggle.

Dataset features:
* Age
* Gender
* BMI
* Number of Children
* Smoking Status
* Region
* Insurance Charges


## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Encoding
5. Train-Test Split
6. Model Training using Linear Regression
7. Model Evaluation
8. Streamlit Deployment



## Installation
Clone the repository:

```bash
git clone https://github.com/your-username/insurance-cost-prediction.git
```

Move into the project folder:

```bash
cd insurance-cost-prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```



## Model Evaluation Metrics
The model is evaluated using:
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score


## Deployment
The application is deployed using Streamlit Cloud.
